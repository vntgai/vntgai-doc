# Glossary

Key terms used across Vantage AI.

| Term | Meaning |
| --- | --- |
| **Backbone universe** | The set of tokens with indexed history that can be **composite-scored**. Tokens outside it still work with the live on-chain tools. |
| **Composite verdict** | The 0–100 fused score (smart-money + CEX supply + OI divergence + concentration) with a colour and pattern label. |
| **Accumulation** | Net supply moving **into** strong hands / **out of** exchanges — bullish. |
| **Distribution** | Net supply moving **to** exchanges / dispersing — bearish. |
| **Smart money** | Wallets scored as informed based on historical behaviour (`wallet_scores`). |
| **CEX supply** | Token balances sitting in centralised-exchange wallets; flows in/out are a positioning signal. |
| **OI divergence** | When coin-denominated **open interest** diverges from price — a derivatives signal. |
| **Sybil cluster** | A group of wallets that look independent but are controlled by one entity. |
| **Funder layer / depth** | How many funding hops back the cluster engine traces to find a shared ancestor. |
| **Token similarity** | Linking wallets that hold near-identical portfolios of other tokens. |
| **Analysis window** | The start–end period (set via the Wallet Cluster price chart) the engine clusters within. |
| **Holdings snapshot** | Balances read at the **end date** of your chosen window (historical, not just current). |
| **Top-10 supply share** | Combined % of supply held by the 10 largest holders — the metric behind holder-change alerts. |
| **Squeeze (k-rule)** | The SqueezePattern4H 4-hour futures pattern behind the Signal-Trade book. |
| **SmartFlow** | On-chain accumulation-building signal (the live paper book). |
| **Signal-Trade** | The public, auto-executed **paper** track record. |
| **Tone (Narrative)** | Risk-on / risk-off / mixed read of the macro tape. |
| **Tier** | Rewards rank: Bronze → Silver → Gold → Platinum → Diamond. |
